# 📡 Customer Churn Prediction (Telecom Industry)

## 📌 Project Overview

This project focuses on predicting customer churn in the telecom industry using Machine Learning.
The goal is to identify customers who are likely to leave and help companies take preventive actions.

## 🎯 Problem Statement

Telecom companies lose customers due to high competition.
It is difficult to identify which customers will churn.

This project helps to:

* Predict customer churn (Yes/No)
* Identify high-risk customers
* Provide actionable business insights


## 📊 Dataset Features

* Customer Info: gender, tenure
* Services: Internet, Streaming, Security
* Billing: MonthlyCharges, TotalCharges
* Target: Churn

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn

## 🔍 Data Processing

* Missing value handling
* Encoding categorical data
* Feature selection

## 🤖 Models Used

* Logistic Regression (Baseline)
* Random Forest Classifier (Advanced)

## 📈 Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 78.8%    |
| Random Forest       | 78.8%    |

## 🧠 Key Insights

* Customers with **high MonthlyCharges** are more likely to churn
* Customers with **low tenure** churn more
* Customers with **low TotalCharges** are less engaged

## 📡 Business Impact

Telecom companies like Jazz & Zong can:

* Identify high-risk customers
* Offer discounts or special packages
* Reduce churn and increase revenue
